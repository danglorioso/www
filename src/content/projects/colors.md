# Colors - Tailwind CSS Color Palette Matcher
A simple tool for finding the closest Tailwind CSS color to any given color input. Supports various color formats including HEX, RGB, HSL, LAB, HTML, and more.

## Features
- Input colors in various formats: HEX, RGB, HSL, LAB, HTML color names
- Find the closest matching color from the Tailwind CSS color palette
- Uses simple Euclidean distance in OKLCH color space for color matching
